Method 1: Add as APK Repository (Recommended)

Add the JamBox repository to your Alpine system:

# Install the jambox-keys package (contains signing key)
# First download and install manually:
wget https://github.com/JamBox-IT/aports/releases/latest/download/jambox-keys-0.2-r0-$(uname -m).apk
apk add --allow-untrusted jambox-keys-*.apk

# Or install the key directly:
wget -O /etc/apk/keys/jambox-65ea124a.rsa.pub \
  https://raw.githubusercontent.com/JamBox-IT/aports/main/JamBox/jambox-keys/jambox-65ea124a.rsa.pub

# Add repository for your Alpine version:
# For Alpine Edge:
echo "@jambox https://jambox-it.github.io/aports/edge/JamBox/$(uname -m)" >> /etc/apk/repositories

# For Alpine v3.19:
echo "@jambox https://jambox-it.github.io/aports/v3.19/JamBox/$(uname -m)" >> /etc/apk/repositories

# Update and install packages
apk update
apk add kanidm@jambox py3-mppsolar@jambox

Method 2: From GitHub Releases

Download individual packages from the Releases page:

# Download for your architecture
wget https://github.com/JamBox-IT/aports/releases/latest/download/<package>-<version>-<arch>.apk

# Install with the key
apk add --allow-untrusted <package>.apk
